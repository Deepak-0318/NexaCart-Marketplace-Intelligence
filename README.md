# NexaCart Marketplace Intelligence

## Overview

NexaCart Marketplace Intelligence is an end-to-end Business Intelligence project focused on identifying the primary operational bottleneck affecting customer satisfaction and marketplace performance.

The project follows an industry-standard analytics workflow, starting from business understanding through data preparation, exploratory analysis, dashboard development, and executive recommendations.

## Tech Stack

- Python
- Pandas
- NumPy
- SQL
- Power BI
- Excel
